# Install dependencies

`cd crm-nextjs`

`yarn`

or

`npm install`

### Install json-server globaly

`npm install -g json-server`
or
`yarn global add json-server`

## Start the project

`json-server db.json --port 4000`

then run

`yarn build`

and

`yarn start`
